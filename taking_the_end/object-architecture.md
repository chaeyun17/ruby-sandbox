# 객체 설계

## Player

- 게임을 하는 사람 플레이어.

### 행위

- 생성: 초기화. 기본 목숨 수 등.
- 목숨 수 조회
- 목숨 차감

### 상태

- 고유 식별자
- 목숨 수

---

## 컴퓨터 Player

- 사용자를 상대하는 컴퓨터 플레이어.

### 행위

- 생성: 초기화
- 끝말잇기 단어 조회

### 상태

- 고유 식별자

---

## Game Context

- 게임 진행 역할

### 행위

- 게임 시작
- 게임 일시정지
- 게임 종료
- 단어 입력: 플레이어가 단어를 입력하면 끝말잇기가 성립하는지 체크. 성공 시, 게임 진행. 실패 시, 목숨 차감 및 재입력 요구

### 상태

- 끝말잇기 단어들
- 플레이어 입력한 단어 갯수
- 제한시간
- 입력 전 측정 시간
- 게임 진행 상태: 진행중, 일시정지, 종료
